# mandelbrot-rs

A simple, interactive, manderbrot set visualizer built on top of [`pixels`](https://github.com/parasyte/pixels)

![demo](screenshots/demo.gif)

## Controls
|           |                     |
|:---------:|:-------------------:|
|Arrow keys | move around         |
|    +      | increase iterations |
|    -      | decrease iterations |
|    x      | zoom in             |
|    z      | zoom out            |
|    c      |  reset              |
